## 1. [The Landmark Verdict That Could Reshape Social Media — ソーシャルメディアを変える歴史的判決](https://www.youtube.com/@CNN10/videos)

**入試テーマカテゴリ**: テクノロジー / 教育・社会問題 / 人権

**出題されやすい学校群**:
- **早慶高（全校）**: SNSと青少年の精神的健康は、早慶の意見文・エッセイ型問題で最も狙われやすいテーマ。賛否両論を読み取る力が問われる。
- **MARCH附属高（青山学院・立教新座）**: 社会問題×テクノロジーの実践的読解として出題可能性あり。

**重要英単語・英語表現**:
| 英語 | 意味 |
|------|------|
| verdict | 判決、評決 |
| lawsuit | 訴訟 |
| addiction | 依存（症） |
| mental health | 精神的健康 |
| intentionally | 意図的に |
| algorithm | アルゴリズム |
| liable | 法的責任がある |
| plaintiff | 原告 |
| regulation | 規制 |
| accountability | 説明責任 |

**背景知識（教養知識）**:
- 2026年3月、カリフォルニア州の陪審がMeta（Facebook/Instagram）とYouTubeに対し、**若者を意図的にSNS依存にさせた**として総額600万ドルの賠償を命じた
- Metaが損害の70%、YouTubeが30%の責任と認定。**「ソーシャルメディアのビッグ・タバコ・モーメント」**（タバコ訴訟に匹敵する転換点）と呼ばれている
- この判決は約2,000件の類似訴訟の「ベルウェザー（先行指標）」となる画期的なケース
- SNSの**アルゴリズム**はユーザーの滞在時間を最大化するよう設計されており、自動再生・無限スクロール・美容フィルターなどが問題視されている
- 米国のCOPPA（児童オンラインプライバシー保護法, 1998年）は13歳未満のデータ収集を規制。EUのDSA（デジタルサービス法, 2022年）はプラットフォーム企業に未成年保護を義務化

## 2. [OpenAI Shuts Down Sora — OpenAIがAI動画生成アプリSoraを停止](https://www.youtube.com/@CNN10/videos)

**入試テーマカテゴリ**: テクノロジー / 人権

**出題されやすい学校群**:
- **関西最難関高（灘・東大寺・西大和）**: AI技術の倫理的課題は、論理的思考力を問う出題に最適。科学×倫理の複合テーマとして出題可能性が高い。
- **早慶高（早大学院・慶應義塾）**: 著作権と技術革新のバランスという高度なテーマで出題される可能性。

**重要英単語・英語表現**:
| 英語 | 意味 |
|------|------|
| artificial intelligence | 人工知能 |
| copyright | 著作権 |
| intellectual property | 知的財産 |
| generate | 生成する |
| criticism | 批判 |
| likeness | 肖像、似姿 |
| shut down | 停止する、閉鎖する |
| ethical | 倫理的な |
| priority | 優先事項 |
| controversy | 論争 |

**背景知識（教養知識）**:
- **Sora**はOpenAIが開発したテキストから動画を生成するAIツール。2024年に発表され大きな注目を集めた
- 停止の理由は**著作権侵害への批判**。著作権者の許可なく学習データに知的財産や有名人の肖像を大量使用していたことが問題化
- **Disneyとの提携交渉も決裂**。コンテンツ産業とAI企業の対立構造が鮮明に
- AI生成コンテンツの著作権問題は世界的な課題。日本でも2023年に文化庁が「AIと著作権」の考え方を公表
- **フェアユース（公正利用）** の概念: 米国著作権法では、教育・批評・研究目的での使用は一定範囲で許容されるが、AI学習データへの適用は法的に未確定

## 3. [USPS Fuel Surcharge — 米国郵便公社が史上初の燃料サーチャージ導入へ](https://www.youtube.com/@CNN10/videos)

**入試テーマカテゴリ**: 経済・グローバリゼーション / 国際関係

**出題されやすい学校群**:
- **慶應系（慶應義塾・慶應志木）**: 戦争→エネルギー価格→物流コストという経済連鎖を読み解く問題として。慶應系は時事的経済テーマを好む。
- **MARCH附属高（中央大学附属・法政大学）**: 身近な物流・郵便サービスへの影響という実生活テーマとして。

**重要英単語・英語表現**:
| 英語 | 意味 |
|------|------|
| surcharge | 追加料金 |
| fuel | 燃料 |
| impose | 課す、導入する |
| postal service | 郵便事業 |
| inflation | インフレーション、物価上昇 |
| go into effect | 施行される、発効する |
| unprecedented | 前例のない |
| logistics | 物流 |
| supply chain | サプライチェーン |
| consumer price | 消費者価格 |

**背景知識（教養知識）**:
- **USPS（United States Postal Service）** は米国の郵便公社。政府機関に近い独立機関で、2024年時点で約63万人の従業員を抱える
- 史上初の燃料サーチャージ（8%）を2026年4月26日から導入。イラン戦争に伴う原油価格高騰が直接の原因
- **燃料サーチャージ**は航空会社や運送会社では一般的だが、郵便公社が導入するのは世界的にも異例
- エネルギー価格と物流コストの関係: 原油価格上昇→輸送コスト増→商品価格上昇→消費者の負担増という連鎖が起きる
- 日本でも2022年以降、ヤマト運輸・佐川急便が燃料サーチャージを導入。エネルギー価格の変動が日常生活に直結する事例

## 4. [Humpback Whale Rescue — ザトウクジラの救出作戦](https://www.youtube.com/@CNN10/videos)

**入試テーマカテゴリ**: 環境問題 / 宇宙・科学

**出題されやすい学校群**:
- **MARCH附属高（全校）**: 動物保護・環境保全のテーマはMARCH附属で最も出題頻度が高い。具体的なストーリー型の読解問題に適合。
- **早慶高（早稲田実業・早大本庄）**: 人間と自然の共生という視点での出題。

**重要英単語・英語表現**:
| 英語 | 意味 |
|------|------|
| humpback whale | ザトウクジラ |
| strand | 座礁する |
| rescue | 救助する |
| excavator | 掘削機（ショベルカー） |
| dredger | 浚渫船 |
| trench | 溝、塹壕 |
| marine life | 海洋生物 |
| conservation | 保全 |
| endangered species | 絶滅危惧種 |
| habitat | 生息地 |

**背景知識（教養知識）**:
- **ザトウクジラ（Humpback whale）** は体長約15m、体重約30トンの大型クジラ。毎年数千kmの回遊を行う
- 1966年に商業捕鯨が禁止される前は絶滅の危機にあったが、保護活動により個体数が回復。2016年にIUCN（国際自然保護連合）のレッドリストで「低懸念」に変更
- クジラの座礁（stranding）の原因: 病気、方向感覚の喪失、海底地形の変化、人工的な水中音波（ソナー）などが考えられている
- 日本は2019年にIWC（国際捕鯨委員会）を脱退し商業捕鯨を再開。捕鯨の是非は国際的な環境・文化論争のテーマ
- **SDGs目標14「海の豊かさを守ろう」** は海洋生態系の保全を掲げている

## 5. [Triceratops Skeleton Auction — 6600万年前のトリケラトプス骨格がオークションに](https://www.youtube.com/@CNN10/videos)

**入試テーマカテゴリ**: 宇宙・科学 / 経済

**出題されやすい学校群**:
- **関西最難関高（灘・東大寺）**: 科学史×倫理の問題として。化石の商業利用と学術研究の対立は論理的思考を問う良い素材。
- **MARCH附属高（明治大学付属明治）**: 科学と社会の接点を考える読解問題として。

**重要英単語・英語表現**:
| 英語 | 意味 |
|------|------|
| fossil | 化石 |
| skeleton | 骨格 |
| auction | オークション、競売 |
| dinosaur | 恐竜 |
| extinction | 絶滅 |
| paleontology | 古生物学 |
| specimen | 標本 |
| investor | 投資家 |
| cryptocurrency | 暗号通貨 |
| heritage | 遺産 |

**背景知識（教養知識）**:
- **トリケラトプス**は約6800万〜6600万年前の白亜紀後期に生息した草食恐竜。3本の角と大きな襟飾り（フリル）が特徴
- 恐竜は約6600万年前の**小惑星衝突（チクシュルーブ衝突体）** により大量絶滅したとされる（K-Pg境界）
- 化石の商業オークションは科学界で論争の的。研究者は「化石は人類共通の遺産であり、博物館に保管されるべき」と主張
- 2020年には**T.レックス「スタン」** が3,180万ドルで落札され、アブダビの博物館に収蔵された
- 今回のトリケラトプスは26歳のシンガポール在住の暗号通貨投資家が出品。科学とビジネスの境界線が問われるケース
