## 1. NASA Artemis II Moon Mission — NASAアルテミスII月面ミッション

**YouTube/Source**: [CNN 10 Podcast (3/27)](https://www.cnn.com/audio/podcasts/cnn-10) | [Transcript](https://transcripts.cnn.com/show/cnc/date/2026-03-27/segment/10)

**入試テーマカテゴリ**: 宇宙・科学技術 / テクノロジー

**出題されやすい学校群**:
- **関西最難関高（灘・東大寺・西大和）**: 科学技術テーマの論説文として最も出題可能性が高い。深い読解力と論理構造の把握が求められる問題に適合。
- **早慶高（早大学院・慶應義塾）**: 宇宙開発の歴史的意義や国際協力の視点から出題される可能性。長文中の事実整理・要旨把握問題として。

**重要英単語・英語表現**:

| lunar | 月の |

**背景知識（教養知識）**:
- **アルテミス計画**はNASAの有人月面探査プログラム。2017年に発表され、アポロ計画（1961-1972）以来約50年ぶりの月面着陸を目指す
- アルテミスIIの乗組員4名には**初の女性（Christina Koch）**と**初のカナダ人（Jeremy Hansen）**が含まれ、宇宙開発における多様性の象徴となっている
- 宇宙開発の歴史: Mercury計画(1958)→Gemini計画(1965)→Apollo計画(1969年月面着陸)→Space Shuttle(1981-2011)→ISS(1998-)→Artemis(2022-)
- アルテミス計画の最終目標は**月面に持続的な拠点を建設**し、将来の**火星有人探査**への足がかりとすること

## 3. Iran Peace Negotiations & Economic Impact — イラン和平交渉と経済への影響

**YouTube/Source**: [CNN 10 Podcast (3/27)](https://www.cnn.com/audio/podcasts/cnn-10) | [Transcript](https://transcripts.cnn.com/show/cnc/date/2026-03-27/segment/10)

**入試テーマカテゴリ**: 国際関係 / 経済・グローバリゼーション

**出題されやすい学校群**:
- **早慶高（慶應義塾・慶應志木）**: 国際政治・経済の総合的理解を問う長文として出題可能性あり。慶應系は時事的テーマを好む傾向。
- **関西最難関高（東大寺・西大和）**: 国際関係の因果関係を論理的に読み解く問題として。

**重要英単語・英語表現**:
| 英語 | 意味 |
|------|------|
| negotiation | 交渉 |

| diplomacy | 外交 |

**背景知識（教養知識）**:
- イランは世界第4位の石油埋蔵量を持つ中東の大国（人口約8,800万人）。1979年のイラン革命以降、米国とは敵対関係にある
- 中東の紛争は世界の**原油価格**に直結する。ホルムズ海峡は世界の石油輸送量の約20%が通過する要衝
- **外交交渉の基本構造**: 和平提案（peace proposal）→ 停戦（ceasefire）→ 和平合意（peace agreement）の段階を踏む
- ダウ平均株価（Dow Jones）は米国の代表的な株価指数で、30社の大企業の株価から算出。地政学リスクが高まると下落する傾向がある
- 戦争が経済に与える影響: エネルギー価格上昇→物価上昇（インフレーション）→消費者の購買力低下→景気後退のリスク

## 4. The "Dirty Dozen" — 農薬と食の安全

**YouTube/Source**: [CNN 10 Podcast (3/26)](https://www.cnn.com/audio/podcasts/cnn-10)

**入試テーマカテゴリ**: 環境問題 / 医療・健康

**出題されやすい学校群**:
- **MARCH附属高（明治大学付属明治・中央大学附属・法政大学）**: 食の安全・健康テーマは中堅難関校で頻出。実生活に即した読解問題として出題されやすい。
- **関西最難関高（洛南・大阪星光）**: 科学的データに基づく論説文として。

**重要英単語・英語表現**:
| 英語 | 意味 |
|------|------|
| pesticide | 農薬 |
| organic | 有機の |
| contamination | 汚染 |
| produce | 農産物 |
| residue | 残留物 |
| exposure | 曝露、さらされること |
| consumer | 消費者 |
| agricultural | 農業の |
| health risk | 健康リスク |
| food safety | 食の安全 |

**背景知識（教養知識）**:
- **Dirty Dozen（汚れた12品目）** は米国の環境保護団体EWG（Environmental Working Group）が毎年発表する、農薬残留量が最も多い12種の農産物リスト
- 常連はイチゴ、ほうれん草、ケール、ブドウ、リンゴなど。逆に農薬が少ない「Clean Fifteen」もある
- **有機農業（organic farming）** は化学農薬・化学肥料を使わない農法。環境負荷が低い一方、収穫量が少なくコストが高い
- 日本の農薬使用量は世界的に見ても多い部類。一方で残留農薬の基準値は厳格に設定されている
- WHO（世界保健機関）は一部の農薬を「発がん性の可能性あり」と分類しており、国際的に規制強化の流れがある

## 5. Hazardous Waste Management — 有害廃棄物の管理

**YouTube/Source**: [CNN 10 Podcast (3/26)](https://www.cnn.com/audio/podcasts/cnn-10)

**入試テーマカテゴリ**: 環境問題

**出題されやすい学校群**:
- **MARCH附属高（全校）**: 環境問題は最も出題頻度の高いカテゴリ。SDGsとの関連で出題される可能性大。
- **早慶高（早稲田実業・早大本庄）**: 環境保全の具体的取り組みについての説明文として。

**重要英単語・英語表現**:
| 英語 | 意味 |
|------|------|
| hazardous waste | 有害廃棄物 |
| landfill | 埋め立て地 |
| disposal | 処分、廃棄 |
| recycle | リサイクルする |
| sustainability | 持続可能性 |
| pollution | 汚染 |
| toxic | 有毒な |
| environmental impact | 環境への影響 |
| waste management | 廃棄物管理 |
| eco-friendly | 環境にやさしい |

**背景知識（教養知識）**:
- **有害廃棄物**には電池、塗料、医療廃棄物、電子機器（e-waste）などが含まれる。通常のゴミとして処分すると土壌・水質汚染を引き起こす
- 世界では年間約5,000万トンの電子廃棄物（e-waste）が発生しており、その多くが途上国に輸出され問題になっている
- **SDGs目標12「つくる責任つかう責任」** は持続可能な生産・消費パターンの確保を掲げ、廃棄物の大幅削減を目指す
- 日本は廃棄物処理法（1970年制定）により、有害廃棄物の処理を厳格に管理。リサイクル率は約20%で、EU諸国（約45%）と比べると改善の余地がある
- **循環型経済（circular economy）** の概念: 「作る→使う→捨てる」の直線型から、「作る→使う→再利用→作る」の循環型への転換が世界的潮流
